Análise  

Com esse código, é possível explorar a correspondência entre os clusters gerados pelo K-Means e os diagnósticos reais, isto é, dignósticos benignos ou malignos do dataset.

Como os clusters correspondem de maneira significativa aos grupos benigno e maligno, é possível concluir que as características celulares analisadas são suficientes para separar os dois tipos de tumor e que o K-Means foi eficaz na identificação dessas diferenças com base nas características celulares fornecidas.
